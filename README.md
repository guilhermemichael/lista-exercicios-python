# Lista de Exercícios Python

Este repositório contém uma lista de exercícios básicos de lógica de programação resolvidos em Python.

O objetivo é praticar conceitos fundamentais como:
- Entrada e saída de dados
- Operações matemáticas
- Estruturas básicas
- Raciocínio lógico

---

##  Exercícios

###  Exercícios básicos

1. Exibir a mensagem "Alô mundo"
2. Mostrar um número informado pelo usuário
3. Soma de dois números
4. Média de 4 notas
5. Conversão de metros para centímetros
6. Cálculo da área de um círculo
7. Área de um quadrado e seu dobro
8. Cálculo de salário mensal
9. Conversão de Fahrenheit para Celsius
10. Conversão de Celsius para Fahrenheit

---

###  Exercícios intermediários

11. Operações com números inteiros e real  
12. Conversão de Gigabytes para Megabytes  
13. Conversão de Gigabytes para Megabytes e Kilobytes  

---

###  Exercícios aplicados

14. Cálculo de multa por excesso de peso (pescador)  
15. Cálculo de salário com descontos (IR, INSS, sindicato)  
16. Loja de tintas (cálculo de latas necessárias)  
17. Loja de tintas com múltiplas opções e otimização  
18. Cálculo de tempo de download  

---

##  Tecnologias utilizadas

- Python 3
- Biblioteca padrão (`math`)
